# 💳 Credit Card Financial Dashboard using Power BI

## 📌 Project Overview

This project is an interactive Credit Card Financial Dashboard developed using **Power BI** and **SQL**. The dashboard analyzes customer transactions, revenue, spending patterns, and key financial KPIs to help businesses make data-driven decisions.

---

## 🎯 Objectives

- Analyze credit card transaction data.
- Monitor business performance using KPIs.
- Understand customer spending behavior.
- Identify revenue trends and transaction patterns.
- Create an interactive dashboard for business insights.

---

## 🛠️ Technologies Used

- Power BI
- SQL
- Power Query
- DAX
- Data Modeling

---

## 📊 Dashboard Features

- Total Revenue KPI
- Total Transactions
- Customer Analysis
- Revenue Trends
- Weekly & Monthly Performance
- Interactive Filters and Slicers
- Dynamic Charts and Visualizations

---

## 📈 Key Insights

- Identified revenue trends across different customer segments.
- Monitored transaction performance over time.
- Analyzed customer spending behavior.
- Built an interactive dashboard for real-time business insights.
